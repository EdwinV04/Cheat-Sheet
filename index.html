HTML & CSS cheat Sheet


Html Tags

Tag Name Description
<p> Paragraph A paragraph always starts on a new line, and is usually a block of text.
<div> Division A division or a section in an HTML document
<br> break Creates a break in the code
<ul> Unordered list The <ul> tag defines an unordered (bulleted) list

CSS Tags

Tag/Name Description
color: Specifies the color needed by user
font-size: Allows the size of the font to change.
font-family: property should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font, and so on
text-align: Proporty will align text to the element that holds it.
